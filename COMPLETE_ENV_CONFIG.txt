# VORNICS WEATHER AI - BACKEND ENVIRONMENT CONFIGURATION

Use these placeholders to configure your Render deployment or local `.env` file. 

> [!IMPORTANT]
> Replace all `your_xxx_here` values with your actual credentials.

---

### 1. CORE APP SETTINGS
```env
PORT=8000
ENVIRONMENT=production
DEBUG=False
SECRET_KEY=your_secret_key_here
ALLOWED_ORIGINS=https://your-frontend-url.onrender.com,http://localhost:3000
```

### 2. FIREBASE (from firestore-key.json)
```env
FIREBASE_PROJECT_ID=your_project_id_here
FIREBASE_PRIVATE_KEY_ID=your_private_key_id_here
FIREBASE_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\nYOUR_PRIVATE_KEY_CONTENT_HERE\n-----END PRIVATE KEY-----\n"
FIREBASE_CLIENT_EMAIL=your_client_email_here
FIREBASE_CLIENT_ID=your_client_id_here
```

### 3. SMTP (For OTP Emails)
```env
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your_email@gmail.com
SMTP_PASSWORD=your_app_password_here
SMTP_FROM_EMAIL=your_email@gmail.com
SMTP_FROM_NAME="Vornics Weather AI"
```

### 4. GITHUB OAUTH
```env
GITHUB_CLIENT_ID=your_github_client_id_here
GITHUB_CLIENT_SECRET=your_github_client_secret_here
```

### 5. GOOGLE OAUTH
```env
GOOGLE_CLIENT_ID=your_google_client_id_here
GOOGLE_CLIENT_SECRET=your_google_client_secret_here
```

### 6. OTHER APIs
```env
OPENWEATHER_API_KEY=your_openweather_api_key_here
VITE_GEMINI_API_KEY=your_gemini_api_key_here
```

### 7. REDIS (Required for Rate Limiting)
```env
# On Render, this will be auto-generated when you add a Redis instance
REDIS_URL=your_redis_url_here
```
